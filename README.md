BEER
====

Getting Started
---------------


To initialize your local repository use

    repo init -u git://github.com/Cl3Kener/BEER.git -b cm-10.1

Then to sync up:

    repo sync

And to start building:

    ./jb-build.sh DEVICE_NAME

Or if you want to sync and build:

    ./jb-build.sh DEVICE_NAME true
